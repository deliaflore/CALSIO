## 🎯 Grading System Implemented

| Score | Grade | GPA |
|-------|-------|-----|
| 80-100 | A | 4.0 |
| 70-79 | B | 3.0 |
| 60-69 | C | 2.0 |
| 50-59 | D | 1.0 |
| 0-49 | F | 0.0 |

**Formulas:**
- Final Score = CA Score + Exam Score
- Overall GPA = Σ(Grade Point × Credit) / Σ Credits
- Average % = Σ(Final Scores) / Number of Courses
